流程：
1、把h5页面通过公众号推送给学生
2、学生访问时在页面上输入学号
3、页面上的js把学号上报给python服务，python服务器根据学号查询出对应的图书报告信息，返回给页面js
4、页面上的js把图书报告信息写入页面中，展示出来
5、学生看到自己的图书报告

组成：
1、annual_report.html 模拟前端h5页面，学生就是在这个上面输入学号，看到图书报告信息
2、server.py 是python服务器，提供一个接口，根据学号返回图书报告信息

使用：
1、用浏览器打开annual_report.html
2、python3 server.py
3、在浏览器里输入学号，点击“查询”
